# Akainu
Akainu Backend API - Express JS Tutorial

## How To Run [FIRST TIME]
- rename `.env.sample` into `.env` and set your configuration
- run `npm install`
- run `sequelize db:create`
- run `sequelize db:migrate`
- run `sequelize db:seed:all`
- run `npm start`
